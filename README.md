# Millwater Consulting — Brand Assets

Public asset repository for the Millwater Consulting brand. Intentionally public and intentionally small: logos, QR code, and social banner only. Everything else — the style guide, ADRs, engineering standards — lives in the private [`mwc-atlas`](https://github.com/millwater-consulting/mwc-atlas) repo.

Brand rules (palette, typography, usage) are defined in the Atlas:

- CSS/HTML surfaces → [`standards/css-html-style-guide.md`](https://github.com/millwater-consulting/mwc-atlas/blob/main/standards/css-html-style-guide.md)
- Slides and documents → [`standards/slides-and-documents-style-guide.md`](https://github.com/millwater-consulting/mwc-atlas/blob/main/standards/slides-and-documents-style-guide.md)

## Usage rules (summary)

- **Do not recolour, rotate, stretch, or add effects** to any logo. Use the variant that already matches your background.
- **Clear space** around any logo ≥ the height of the "M" in "Millwater".
- **Minimum size:** wordmark ≥ 0.75" / 120 px wide; square ≥ 0.5" / 64 px.
- **Never** place the teal-gradient variants on busy photography. If you need a transparent variant, request one — don't cut one out yourself.
- **QR code:** do not regenerate or re-style. Use the approved PNG as-is so the embedded "M" and palette remain consistent.

Full rules: see the Atlas guides linked at the top.
